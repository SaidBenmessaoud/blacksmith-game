Blacksmith Game

Blacksmith is a simple text-based game where you play as a blacksmith. You can make weapons using ore, wood, and fire. You will sell them for gold and use that gold to buy more ore and wood to make more weapons. Use the following commands to interact with the game:

  • buy(item): Buy supplies (wood or ore)
  • make(item): Make items (sword or axe)
  • sell(item): Sell items (sword or axe)
  • fire(): Start or stop the fire
  • inventory(): Show current inventory
  • help(): Show game instructions
